HypergraphPlotter Library Demo

This demo showcases the HypergraphPlotter class library imported as an ES module, mounting an interactive graph simulation inside a custom container and updating options dynamically.

Integration Code Example

import { HypergraphPlotter } from './hypergraph-plotter.js';

const plotter = new HypergraphPlotter('#plotter-mount', {
  layoutType: 'spring-embedding',
  edgePalette: 'warm-sunset',
  showGrid: true
});

plotter.setData({
  vertices: [
    { id: 'v1', label: 'A' },
    { id: 'v2', label: 'B' },
    { id: 'v3', label: 'C' }
  ],
  hyperedges: [
    { id: 1, vertices: ['v1', 'v2', 'v3'] }
  ]
});
